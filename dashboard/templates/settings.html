Environment Variables

Settings are loaded from the .env file in the project root. Edit that file to change API keys and configuration.

TELEGRAM_BOT_TOKEN Telegram bot authentication
AI_PROVIDER openai / anthropic / local
AI_API_KEY AI provider API key
AI_MODEL Model to use for generation
GITHUB_TOKEN GitHub API access
DASHBOARD_PORT Web dashboard port (default: 5000)
